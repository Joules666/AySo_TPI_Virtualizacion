## Descripción General

Este repositorio contiene el material correspondiente al **Trabajo Práctico Integrador (TPI)** sobre **Virtualización con Oracle VirtualBox y Kali Linux**.  
El objetivo principal fue **instalar, configurar y probar un entorno virtualizado** utilizando **Oracle VirtualBox** como *hypervisor tipo 2*, con una máquina virtual basada en **Kali Linux**.

Dentro del entorno virtual se verificó la instalación de **Python** y se ejecutó un **script propio en lenguaje Python** que realiza la conversión de números decimales a binarios y viceversa.
